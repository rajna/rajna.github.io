2、os安装ipython
1）、sudo pip install ipython若报错
2）、pip install --upgrade setuptools，或者试试sudo -H pip install ipython --ignore-installed six，再或者试试
sudo -H pip install ipython --ignore-installed numpy如果还是报错，那么可能是由于系统的保护
3）、解除系统保护方法如下链接http://chaishiwei.com/blog/994.html
4）、pip install ipython或者pip install "ipython[all]"
5)、pip install -U ipython
6）、sudo pip install pync
7）、sudo -H pip install jupyter --ignore-installed six
8）、ipython notebook
9）、关闭jupyter网页之后还需要再终端输入control+c才算真正的把notebook关闭了。
10）、安装matplotlib包 ：sudo -H pip install matplotlib --ignore-installed six
